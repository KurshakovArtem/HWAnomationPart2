# Домашнее задание: Animations — создание анимаций. Часть 2

1. Пользователь может не увидеть 100 повторений, потому что анимация привязана к ViewHolder(переизпользоваемому), который уходит с экрана при скроле(или при пересоздании Activity при повороте или навигации), анимация или прерывается или продолжается вне экрана.
2. Анимация привязана к View а не к данным поста. При переиспользовании карточки поста, анимация переходит к новому посту, а View остаётся тем же, но перезаписанным.
